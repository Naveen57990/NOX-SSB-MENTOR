A hybrid app to stimulate PSYCHOLOGICAL TESTS of ssb and provide ai based feedback highlighting strengths,weakness and officer like qualities(OLQs).

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
